# Markers in .NET MAUI Spark Charts

Markers can be used to highlight individual data points in a spark chart. They can be enabled by setting the `ShowMarkers` property to `true`. The shape and color of the markers can be customized using the `MarkerBrush` and `MarkerShape` properties.
